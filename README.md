# SCT_CS_03

Password Strength Checker
Assess password strength based on length, character variety, and security criteria.
Requirements
Python 3.x (no external libraries needed)
Usage
bashpython password_strength_checker.py
Features

Scores passwords from 0 to 12 points
Checks: length, uppercase, lowercase, numbers, special characters
Provides a visual strength meter and recommendations
Includes password security tips
